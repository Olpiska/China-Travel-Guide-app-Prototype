AI-Based China Travel Assistant 🇨🇳

An AI-powered, multilingual travel recommendation system designed for international visitors traveling to China. This project is developed as a web-based interactive prototype using Python and Streamlit.


🔍 Overview


This application provides personalized travel recommendations based on user preferences such as budget, travel duration, interests, and travel style. The system integrates multiple AI-driven modules into a single unified assistant.
Unlike traditional travel applications, this system focuses on:

- Personalization

- Cultural adaptability

- Multilingual interaction

- Budget-aware planning


⚙️ Features


🧠 AI Travel Recommendation System

Suggests cities based on:
 - Budget

 - Travel duration

 - User interests

 - Preference for popular vs. quiet destinations

Provides:

 - Tourist attractions

 - Local markets

 - Cultural insights

 - Food recommendations

 - Average daily expenses


🗺️ Travel Planner


- Generates a day-by-day travel plan

- Suggests where to go each day

- Helps optimize time and budget


🌍 Multilingual AI Assistant


Supports multiple languages:

- English

- Turkish

- Russian

- Arabic

- German

- Spanish

- Chinese

- Polish

- French

The system dynamically adapts responses and budget calculations based on language.


💰 Smart Budget System


- User defines budget and currency

- System adapts recommendations accordingly

- Provides estimated daily costs


🕌 Cultural & Lifestyle Preferences


Includes options such as:

- Muslim-friendly / Halal preferences

- Adjusts food and environment recommendations accordingly


🧭 Exploration Module


Users can explore detailed information about selected cities:

- Food culture

- Traditions and local customs

- Clothing and regional differences

- Language insights


🏨 Accommodation Suggestions


- Recommends hotels and hostels

- Matches options with user budget


🚨 Travel Support Information


- Public holidays in China

- Emergency contact numbers

- Essential travel apps


🖥️ System Architecture


- Built with Python

- Interface developed using Streamlit

- Modular AI-based decision system

- Rule-based + preference-based recommendation logic


🚀 Purpose of the Project


This project aims to demonstrate how artificial intelligence can be used to improve user experience in travel planning through:

- Intelligent recommendation systems

- User-centered design

- Cross-cultural adaptability

It is designed as a research-oriented prototype, with potential for future expansion into a full-scale application.
